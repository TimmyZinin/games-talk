04 04 · Motorlar / Engines

Üç motor.
Üç farklı iş.

EN The choice of engine decides everything that comes after — load time, bundle size, what AI tools you can plug in, even how you ship. Pick by output, not by hype.

TR Motor seçimi sonraki her şeyi belirler — yükleme süresi, paket boyutu, hangi yapay zekâ araçlarını entegre edebileceğin, hatta nasıl yayınlayacağın. Hype'a göre değil çıktıya göre seç.

For lead-magnet web games

Three.js

5M downloads/week. Bundle 0.5–1 MB. Loads in 2–6 seconds on mobile. The default for browser-first 3D — landing page games, interactive sites, narrative walkers like Afterhumans.

TR Tarayıcıda 3D için varsayılan. Lead magnet'ler, interaktif siteler, Afterhumans gibi anlatı yürüyüşleri için.

OutputWeb
Bundle~ 1 MB
LicenseFree · MIT
AI fitClaude · Cursor
For polished cross-platform

Unity

The industry default. Unity 6 + Unity AI Generators (sprites, textures, materials, animation, sound). WebGL builds 25–50 MB and 8–30 s to load — heavy for marketing, perfect for distributed builds.

TR Endüstri standardı. Unity 6 + Unity AI üreticileri. WebGL paketleri 25–50 MB; pazarlama için ağır, oyun mağazası için ideal.

OutputWin · Mac · iOS · Android · Web · Console
Bundle25–50 MB WebGL
LicenseFree up to revenue cap
AI fitTripo · Meshy · Suno · Claude C#
For photoreal & high-end

Unreal 5 / 6

Photorealism, Nanite micro-geometry, Lumen lighting, MetaHuman. Steeper learning curve, larger team requirement, but no rival when you need film-grade output. We don't ship lead magnets here — we use it for high-end product visuals.

TR Fotogerçekçilik, MetaHuman, sinema kalitesinde çıktı. Öğrenme eğrisi daha dik, ekip gerektirir. Lead magnet için değil, premium ürün görselleri için kullanırız.

OutputPC · Console · Film · VR
BundleHeavy / native

Tool 01 / Code

Claude Code

EN Anthropic's terminal-native coding agent. The largest context window on the market — drop a whole Unity project in and it remembers it. Best for refactors, generating C# scripts, shaders, gameplay code.

TR Anthropic'in terminal tabanlı kod ajanı. Piyasadaki en büyük bağlam penceresi. Refactor, C# script ve oynanış kodu üretmek için en iyisi.

→ Used in Marina, Kloaka, Afterhumans

In Marina Claude wrote the entire state machine, the funnel, the win/lose, and the 4th-wall finale. In Kloaka it wrote enemy AI, the upgrade system and the HTML overlay CTA. This is my main tool — it ran every game on this page.

claude.com/code CLI agent Pro $20/mo
Tool 02 / Code

Cursor

EN The IDE I open every morning. Multi-file edits, tab-completions, agent mode, fast loop. Where Claude Code is the heavy refactor tool, Cursor is the daily driver — small edits, surgical fixes, exploring an unfamiliar codebase.

TR Her sabah açtığım IDE. Çok dosyalı düzenleme, tab tamamlama, ajan modu. Claude ağır iş, Cursor günlük sürücü — küçük değişiklikler, cerrahi düzeltmeler.

→ Daily driver across all 4 games

For Three.js work in Afterhumans I lived in Cursor — fast preview cycles, AI tab completions, multi-file refactors. Combined with Claude Code, this is the pair that ships.

cursor.com IDE agent Free tier
Tool 03 / Review

Codex CLI

EN OpenAI's read-only code review agent. It runs in a sandbox, can't write files, can't lie about what's there. I use it as a second pair of eyes after Claude finishes a sprint — and it's caught fabricated assets, drifted READMEs, inflated self-grades.

TR OpenAI'nin salt-okunur kod inceleme ajanı. Sandbox'ta çalışır, yazmaz, yalan söyleyemez. Claude bir sprint bitirince ikinci göz olarak kullanırım.

→ Caught the inflated grades in Afterhumans

In Afterhumans Codex's audit found that "9.2/10 sprint" reports were really 7.8 — missing photo_nikolai.png, drifted art-direction docs, dual HITL skipped. Without it I would have polished a fiction.

github.com/openai/codex Read-only sandbox ChatGPT Plus
Tool 04 / 3D

Tripo 3D

EN Text or image into a game-ready 3D mesh in 60 seconds. Quad topology, GLB / FBX / USDZ export. The model walks straight into Unity or Three.js without a Blender step. Free tier: 600 credits / month.

TR Metinden veya görselden 60 saniyede oyuna hazır 3D model. Quad topoloji, GLB/FBX/USDZ. Blender adımı olmadan doğrudan Unity'ye girer.

→ 11 enemy & boss meshes for Kloaka Pechali

Every enemy in Kloaka — the yellow drone-CV from HH.ru, the LinkedIn vampire, the HR-bog frog, the final-boss interviewer — started as a Tripo prompt. 14 models attempted, 11 shipped, 3 placeholders. 26 MB total after gltfpack compression.

tripo3d.ai GLB · FBX · USDZ Free 600 cr/mo
Tool 05 / 3D

Meshy AI

EN Tripo's main rival. The most mature ecosystem — first-party plugins for Unity, Blender and Unreal, clean topology, retexturing flow. If you're committing to a single 3D-mesh AI for a studio, this is the one for plugin maturity.

TR Tripo'nun ana rakibi. En olgun ekosistem — Unity / Blender / Unreal için yerleşik eklentiler. Bir stüdyo için tek 3D-mesh aracı seçecekseniz, plugin olgunluğu Meshy.

→ Studied as a Tripo replacement

I tried Meshy as an alternative when Tripo started rate-limiting on free tier. Output was equally usable, the Unity plugin saved a copy-paste step. For solo work I stayed on Tripo for cost; for a team I'd default to Meshy.

meshy.ai Unity / Blender / Unreal plugins Free 200 cr/mo
Tool 06 / 3D

Rodin · Hyper3D

EN The premium tier of AI 3D. Gen-2 model — photorealism, 4K PBR textures, 10B parameters. Slower, paid, heavier — but the best output of the three when realism matters. The choice when Tripo / Meshy looks "cartoony".

TR Yapay zekâ 3D'nin premium katmanı. Gen-2: fotogerçekçilik, 4K PBR. Daha yavaş, ücretli, ağır — ama gerçekçilik istendiğinde üçünün en iyisi.

→ Reserved for Antaria / cTrader visuals

Not used in any of these games — game art looks better stylised. But for Antaria Homes 3D property visuals and cTrader dashboard renders this is where I go for production-grade output.

hyper3d.ai 4K PBR Free tier
Tool 07 / Worlds

Marble · World Labs

EN Fei-Fei Li's first commercial product, launched November 2025. Take a photo or write a prompt, get back a walkable 3D world — Gaussian splats plus mesh export. You can drop the result straight into Three.js or Unity.

TR Fei-Fei Li'nin ilk ticari ürünü, Kasım 2025'te yayınlandı. Bir fotoğraf veya prompt — gezilebilir 3D dünya. Three.js'e ya da Unity'ye doğrudan girer.

→ Next experiment for Afterhumans worlds

My next game build will not have a 3D modeller in the pipeline at all. The grey room in Afterhumans is what Marble is built for — and Marble exports the same GLB I was building by hand.

worldlabs.ai Gaussian splats + mesh Freemium
Tool 08 / Worlds

Genie 3

EN Google DeepMind's playable-world model. 720p, 24 fps, ~60 seconds to generate, navigable in real time. The shape of where games are going: write a sentence, walk through the result on the spot, decide if it's the world you wanted.

TR DeepMind'ın oynanabilir dünya modeli. 720p, 24fps, ~60 saniyede üretiliyor, gerçek zamanlı gezilebilir. Oyunların gittiği yön: bir cümle yaz, içinde yürü, beğen ya da değiştir.

→ Reference point for "what's next"

Currently US-only via AI Ultra. We can't ship a Genie 3 game tomorrow — but it's the reference for what every world-AI tool will look like by the end of 2026. Marble is the practical version today, Genie 3 is the public preview of the future.

deepmind.google/models/genie Real-time worlds AI Ultra · US
Tool 09 / Worlds

Mirage · Decart

EN Real-time video-to-video world transformation, 20 fps. Stream a webcam feed in, get back a different world out — with rooms, lighting, weather all changed. Decart's Magica engine is the next iteration: AI-native UGC engine where you modify the world in natural language while you play it.

TR Gerçek zamanlı video-video dünya dönüşümü, 20 fps. Webcam akışını al, farklı bir dünya çıkar. Magica motoru bir adım sonra: oynarken doğal dilde dünyayı değiştir.

→ The "live world" demo for the room

If we have stage time and the Wi-Fi cooperates, this is the demo I'd run live. Mirage is the only AI tool where the audience watches the world change in real time as I type — not "compute, then result", but compute as result.

mirage.decart.ai Real-time video2video Freemium

Pollinations · FLUX

EN Free, no-account, URL-based image generation API around FLUX. Drop a prompt into a URL, get back a JPG. No keys, no quotas in practice. Made for fast event cards, OG images, mood pieces — not high-fidelity hero art.

TR Hesapsız, ücretsiz, URL tabanlı FLUX API. URL'ye bir prompt koy, geri JPG al. Hız için, yüksek hassasiyet için değil.

→ 11 event cards · OG images for Kloaka & Marina

Every event in Marina has a Pollinations photo. The OG image of Kloaka Pechali on social previews is a single Pollinations call. When something needs an image and "good enough" is good enough — this is the cheapest fast path I know.

06 06 · Game Design Document

GDD.
Altı bölüm. O kadar.

EN A GDD is not a 200-page bible. In 2026 it's a living markdown file you write with Claude. Done well, it makes every later prompt 10× sharper. Done badly, you'll prompt the same things at every session.

TR GDD 200 sayfalık kutsal kitap değildir. 2026'da Claude ile birlikte yazdığınız canlı bir markdown dosyasıdır. İyi yazılırsa sonraki promptların 10 kat keskin olur.

01Pitch

Name, genre, platform, core loop in a single sentence. If you can't say it in one line, you don't have a game yet.

02Mechanics

Actions, controls, progression, fail / win. What does the player do, and what does the game do back?

03Art

Style, references, mood board, palette. AI texture & mesh tools work much better when this section is real.

04Narrative

World, characters, tone, pacing. Even an arcade has tone — Kloaka's tone is "office despair with rocket launchers".

05UI & Audio

Wireframes, music brief, SFX list. Suno gets sharper output from a brief than from a feeling.

06Scope

MVP, monetization, ship date — and what's out. The "out" list is the most valuable line in the document.

Pro tip — after writing the GDD, ask Claude one line: "Push back on this GDD. Where will I get stuck in week two?"

07 07 · Workflow

Pazartesi fikir.
Cuma yayında.

EN Five steps. Each step is a prompt and a check. Codex reviews what Claude builds. The web is the platform. Hosting is GitHub Pages — free, fast, custom domain.

TR Beş adım. Her adımda bir prompt ve bir kontrol. Codex, Claude'un yazdığını inceler. Platform web. Barındırma ücretsiz: GitHub Pages.

01 · Idea→ 02 · GDD with Claude→ 03 · Plan with Codex→ 04 · Build with Claude Code→ 05 · Assets · Tripo · Suno · FLUX→ 06 · Ship to web

Same loop scales from a 200-line web toy to a 4-level Unity build. The bottleneck is taste & decisions, not typing speed.
