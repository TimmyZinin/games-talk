05 / Architecture · Mimari
USER → AGENT → LLM

Üç katman. Sihir orta katmanda.

EN You don't type prompts straight into a language model. Between you and the model sits a coding agent — the interpreter. It reads your intent, picks tools, edits files, runs commands, checks results, and only then asks the LLM what to do next.

TR Promptu doğrudan dil modeline yazmıyorsun. Aranızda bir kod ajanı var — yorumlayıcı. Niyetini okur, araç seçer, dosya düzenler, komut çalıştırır, sonucu kontrol eder, ve ondan sonra LLM'e ne yapılacağını sorar.

👤

USER

You. Natural language. Pictures. Voice. A single sentence:
"Make a 4-level WebGL shooter."

Sen. Doğal dil.

⚙

AGENT · INTERPRETER

Claude Code · Codex · Cursor · Aider. Reads intent. Picks tools (read · write · bash · web · MCP). Edits files. Runs commands. Loops until done.

parse intent → pick tool → execute → check → ask LLM → repeat

Yorumlayıcı. Niyeti okur, araç seçer, dosya düzenler, döngüye girer.

🧠

LLM

Claude · GPT · Gemini. Pure language model. Doesn't see your disk. Doesn't run code. Only answers: "here's what to do next."

Saf dil modeli. Diskini görmez. Sadece "şimdi ne yap" der.

⇄⇄
The agent is what makes AI useful for code. Pick a model and you pick intelligence. Pick an agent and you pick how that intelligence behaves on your machine.

06 / Game Design Document
ALTI BÖLÜM · O KADAR

GDD. Six sections. Max.

EN Not a 200-page bible. In 2026 it's a living markdown you write with Claude. Done well, every later prompt is 10× sharper. Done badly, you re-prompt the same things every session.

TR 200 sayfalık kutsal kitap değil. 2026'da Claude ile birlikte yazdığın canlı markdown.

01

Pitch

Name, genre, platform, core loop in one sentence.

02

Mechanics

Actions, controls, progression, fail / win.

03

Art

Style, references, mood board, palette. AI tools work better here.

04

Narrative

World, characters, tone, pacing.

05

UI & Audio

Wireframes, music brief, SFX list.

06

Scope

MVP, monetization, ship date — and what's OUT.

Pro tip — ask Claude: "Push back on this GDD. Where will I get stuck in week two?"
